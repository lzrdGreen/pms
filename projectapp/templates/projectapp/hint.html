Project Management System (PMS) App

Purpose

The PMS app is designed to provide a user-friendly, versatile project management solution. It allows users to efficiently create, view, edit, and delete tasks and projects (CRUD operations), offering both list and board views for project visualisation. The app aims to streamline project management processes while maintaining simplicity and usability.

Key Features

  • Dual view options: list and board views for projects

  • Project timeline visualisation with task dependencies

  • Search functionality across projects and tasks

  • Intuitive user interface

  • Streamlined project management workflows to enhance productivity

  • Task and project CRUD operations

  • Content creation and editing according to user permissions

  • Task prioritisation and categorisation

Core Philosophy

Simplicity, usability, and added value are at the heart of the PMS app. The focus is on delivering essential features that enhance project management without overwhelming users with complexity.

Major Directions for Further Development

Basic Collaboration Tools

  • Implement social media style user profiles
  • Enable seamless assignment of tasks to specific team members
  • Add basic file-sharing functionality with image uploads
  • Develop a simple in-app notification system for task assignments and deadlines

Visual Project Management

  • Introduce a board view with customisable columns
  • Add progress bars for projects

Advanced Content Creation

  • Integrate a lightweight text editor for formatting
  • Enhance categorisation based on complex tagging systems
  • Improve search functionality with filters

Time Management Features

  • Implement a calendar view for deadlines and milestones
  • Add basic time tracking for tasks
  • Automatically create recurring tasks for repetitive jobs
  • Develop a simple dashboard for upcoming deadlines and task assignments

Customisation and Flexibility

  • Allow users to create custom project templates
  • Enable basic task and project export functionality

Each of these directions can be further developed based on client needs and budget constraints, prioritising simplicity in implementation and cost-effectiveness in development.
